一篇较好的mysql语句总结文档

基础内容

MySQL基础命令

  • 默认端口号：3306
  • 查看服务器版本：select version(); 或者 cmd命令 mysql -verison
  • 登录数据库：mysql -uroot -p
  • 退出数据库：exit/quit
  • 查看当前系统下的数据库：show databases;
  • 创建数据库：create 库名;
  • 使用数据库：use 库名;
  • 查看表：show tables;
  • 建表：create table 表名 (字段名 + 空格 + 数据类型);
  • 查看表结构：desc 表名;
  • 添加值：insert into 表名 (列名) values (值);
  • 查看表中所有数据：select * from 表名;
  • 查询建表时的结构：show create table 表名;
  • 删除字段中的值：delete from 表名 where 条件;
  • 删除表中的字段：delete from 表名 drop column 字段名; 或alter table 表名 drop 字段名
  • 删除表：drop table 表名;
  • 删除库：drop database 库名;
  • 主键约束：primary key
  • 唯一约束：unique
  • 非空约束：not null
  • 默认约束：default
  • 外键约束：foreign key（外键）references主表（主键）
  • 查看别的数据库的表格：show tables from 表名;

常用语句

  • mysql 连接MySQL数据库的命令，需要指定用户名和密码。
  • use 使用某个数据库。
  • show databases 显示所有数据库。
  • create database 创建新的数据库。
  • drop database 删除数据库。
  • create table 创建新的表。
  • alter table 修改表结构。
  • drop table 删除表。
  • insert into 向表中插入新的记录。
  • delete from 删除表中符合条件的记录。
  • update 更新表中符合条件的记录。
  • select 查询表中的记录。
  • where 对查询的记录进行条件筛选。
  • order by 对查询的记录进行排序。
  • group by 对查询的记录进行分组。
  • having 对分组后的记录进行筛选。
  • set names 设置字符集编码。

一些基础知识

  • sql语句需要以分号；结尾
  • 且可以同时运行多个任务
  • sql语句无大小写区分
  • 忽略空白和空行，可以放置在多个文本行上或者多个语句在一个行上组合
  • 主键是表中唯一标识表记录的字段，每行必须包含唯一值，不能存在NULL,（例如统计个人信息，名字是必须的，这是能确定身份的绝对信息）。
  • 各种运算可使用()划分优先级
  • 文本内容需要用英文单引号包括
  • 子查询:嵌套使用select语句，利用（）区分优先级
  • select操作均为副本操作

SQL约束

  • not null表示不能包含任何null值
  • 唯一unique表示不允许在表中插入重复值
  • 主键primary key表示强制表接受特定列的唯一数据，此约束创建一个唯一索引以更快的访问表
  • 检查check表示根据逻辑表达式确定值是否有效
  • 默认值default表示在将数据插入表中时如果没有向列提供值，则该列获得设置为默认值的值
  • auto_increment自动递增 SQL约束用法为 name varchar（100）not null，即在创建时跟在列后面

运算符

算术运算符

运算符 描述 示例 结果
+ 加法 SELECT 10 + 5; 15
- 减法 SELECT 10 - 5; 5
* 乘法 SELECT 10 * 5; 50
/ 除法 SELECT 10 / 5; 2
% 或 MOD() 取模（余数） SELECT 10 % 3; 1
DIV 整数除法（MySQL） SELECT 10 DIV 3; 3

比较运算符

运算符 描述 示例 结果
= 等于 SELECT 10 = 10; TRUE
!= 或 <> 不等于 SELECT 10 != 5; TRUE
> 大于 SELECT 10 > 5; TRUE
< 小于 SELECT 10 < 5; FALSE
>= 大于等于 SELECT 10 >= 10; TRUE
<= 小于等于 SELECT 10 <= 5; FALSE
<=> NULL安全的等于（MySQL） SELECT NULL <=> NULL; TRUE
BETWEEN 在范围内 SELECT 5 BETWEEN 1 AND 10; TRUE
NOT BETWEEN 不在范围内 SELECT 15 NOT BETWEEN 1 AND 10; TRUE
IN 在列表中 SELECT 5 IN (1,3,5,7); TRUE
NOT IN 不在列表中 SELECT 2 NOT IN (1,3,5,7); TRUE
IS NULL 是NULL值 SELECT NULL IS NULL; TRUE
IS NOT NULL 不是NULL值 SELECT 10 IS NOT NULL; TRUE
LIKE 模式匹配 SELECT 'abc' LIKE 'a%'; TRUE
NOT LIKE 模式不匹配 SELECT 'abc' NOT LIKE 'd%'; TRUE

逻辑运算符

运算符 描述 示例 结果
AND 或 && 逻辑与 SELECT TRUE AND FALSE; FALSE
OR 或 | 逻辑或 SELECT TRUE OR FALSE; TRUE
NOT 或 ! 逻辑非 SELECT NOT TRUE; FALSE
XOR 逻辑异或 SELECT TRUE XOR FALSE; TRUE

位运算符

运算符 描述 示例 结果
& 位与 SELECT 5 & 3; 1
| 位或 SELECT 5 | 3; 7
^ 位异或 SELECT 5 ^ 3; 6
~ 位取反 SELECT ~1; -2
<< 左移位 SELECT 1 << 2; 4
>> 右移位 SELECT 8 >> 2; 2

字符串运算符

运算符 描述 示例 结果
| 或 CONCAT() 字符串连接 SELECT 'Hello' | ' World'; 'Hello World'
LIKE 模糊匹配 SELECT 'abc' LIKE 'a%'; TRUE
NOT LIKE 模糊不匹配 SELECT 'abc' NOT LIKE 'd%'; TRUE
REGEXP 正则匹配 SELECT 'abc' REGEXP '^a'; TRUE
NOT REGEXP 正则不匹配 SELECT 'abc' NOT REGEXP '^d'; TRUE

集合运算符

运算符 描述 示例
UNION 合并结果集（去重） SELECT a FROM t1 UNION SELECT b FROM t2;
UNION ALL 合并结果集（不去重） SELECT a FROM t1 UNION ALL SELECT b FROM t2;
INTERSECT 交集（共同部分） SELECT a FROM t1 INTERSECT SELECT b FROM t2;
EXCEPT 或 MINUS 差集（A有B无） SELECT a FROM t1 EXCEPT SELECT b FROM t2;

赋值运算符

运算符 描述 示例
:= 赋值 SET @var := 10;
= 赋值（在SET语句中） SET @var = 10;

特殊运算符

运算符 描述 示例
CASE 条件表达式 CASE WHEN condition THEN result ELSE default END
COALESCE() 返回第一个非NULL值 SELECT COALESCE(NULL, NULL, 'default');
NULLIF() 相等返回NULL SELECT NULLIF(10, 10);
IFNULL() 为NULL则返回替换值 SELECT IFNULL(NULL, 'replacement');

运算符优先级（从高到低）

优先级 运算符
1 ()（括号）
2 -（一元负号）, ~（位取反）
3 *, /, %, DIV
4 +, -（减法）
5 <<, >>
6 &
7 |
8 =, !=, <>, <=>, <, <=, >, >=, IN, IS, LIKE, REGEXP
9 BETWEEN, CASE, WHEN, THEN, ELSE
10 NOT
11 AND, &&
12 XOR
13 OR, |
14 :=（赋值）

常用函数运算符

类别 运算符/函数 描述 示例
聚合函数 COUNT() 计数 SELECT COUNT(*) FROM users;
聚合函数 SUM() 求和 SELECT SUM(salary) FROM employees;
聚合函数 AVG() 平均值 SELECT AVG(score) FROM grades;
聚合函数 MIN() 最小值 SELECT MIN(price) FROM products;
聚合函数 MAX() 最大值 SELECT MAX(age) FROM users;
字符串函数 CONCAT() 连接字符串 SELECT CONCAT(first_name, ' ', last_name);
字符串函数 SUBSTR() 子字符串 SELECT SUBSTR('Hello', 1, 3);
字符串函数 LENGTH() 字符串长度 SELECT LENGTH('abc');
字符串函数 UPPER() 转大写 SELECT UPPER('hello');
字符串函数 LOWER() 转小写 SELECT LOWER('HELLO');
日期函数 NOW() 当前时间 SELECT NOW();
日期函数 CURDATE() 当前日期 SELECT CURDATE();
日期函数 DATE_ADD() 日期加法 SELECT DATE_ADD(NOW(), INTERVAL 1 DAY);
日期函数 DATEDIFF() 日期差 SELECT DATEDIFF('2024-01-01', '2023-01-01');

模式匹配通配符

通配符 描述 示例
% 匹配0个或多个字符 LIKE 'a%' 匹配以'a'开头的字符串
_ 匹配单个字符 LIKE 'a_' 匹配以'a'开头且长度为2的字符串
[] 匹配指定字符集（某些数据库） LIKE '[abc]%' 匹配以a、b或c开头的字符串
[^] 不匹配指定字符集（某些数据库） LIKE '[^abc]%' 匹配不以a、b、c开头的字符串
