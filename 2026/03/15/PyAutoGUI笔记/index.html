PyAutoGUI笔记

发表时间：2026-03-15 阅读： 评论：

计算机基础 python

补充： 该技术与PyNput技术关联较高，主要为操作与识别特化。

通用功能

  1. 暂停.PAUSE，默认为0.1秒
  2. 临时休眠.sleep()
  3. 倒计时countdown()
  4. 自动防故障.FAILSAFE。鼠标划到左上角（0.0）会强制停止pyautogui程序，默认为True

一、鼠标控制

**1.位置操作

  1. .moveTo(x,y,duration=time),duration参数为设置移动时间，默认为0，即瞬间移动，改函数为移动到（x,y）
  2. .move(x,y,duration=time),该函数为相对当前坐标移动x和y。
  3. .position()，获取鼠标指针当前位置

**2.点击操作

  1. .click(x,y,button="key")，单击,不输入坐标默认当前位置单击，button参数为设置"left"，"right"，"middle"三个键，或者直接使用.leftClick() .rightClick() .middleClick()
  2. .doubleClick() 双击
  3. .tripleClick 三击
  4. .mouseDone() 按下，.mouseUp() 松开

3.拖动操作

  1. .dragTo(x,y) 绝对位置操作
  2. .drag(x,y) 相对位置操作
  3. .scroll(int) 滚动操作
函数 介绍
moveTo(x,y,duration) 将鼠标指针立即移动到屏幕的指定位置,耗时 duration 秒
move(dx,dy,duration) 相对于"当前的位置"移动鼠标指针,耗时 duration 秒
position() 获取鼠标位置
click(x,y,clicks,button) 在 (x, y) 处点击鼠标 button 键 clicks 次
leftClick(x,y) 左击
rightClick(x,y) 右击
middleClick(x,y) 中击
doubleClick(x,y) 双击
dragTo(x, y) 拖动鼠标，绝对位置
drag(x, y) 拖动鼠标，相对位置
scroll(offset) 滚动鼠标，传递正整数表示向上滚动， 传递负整数表示向下滚动

二、键盘控制

输入

  1. .write("string",duration) 输入键名字符串，这些操作产生什么效果， 取决于当前获得焦点的窗口和文本输入框。 可能需要先向文本框发送一次鼠标单击事件， 确保它获得焦点。对于A和!这种字符，会自动模拟按住Shift键。需要注意
  2. .press('key',time) 按住按键key持续time秒
  3. .keyDone('key') .keyUp('key') 按住和松开按键
  4. 组合键功能： 例如Ctrl+C，可以调用两次keyDone()，然后调用keyUp松开按键。
  5. .hokey('key_1','key_2',...) 可同时接受多个键字符串，按顺序按下，再按照相反顺序松开。
  6. 键名： 在PyAutoGUI中， 一些特俗的键表示为短的字符串： 'esc’表示Esc键， 'enter’表示Enter键，'win'表示windows键，上下左右箭头为对应英文单词。调用.KEY_NAMES可以查看特殊按键列表。

三、屏幕图像处理

1.获取屏幕

  1. size = size() 获取屏幕尺寸，返回两个整数的元组，不收屏幕缩放比例影响。
  2. im = screenshot() 全屏截图
  3. im.save("save_path") 保存
  4. im = screenshot("save_path",region=(x_1,y_1,x_2,y_2)) 截图并保存指定区域到路径

2.像素与匹配

  1. pix = pixiv(x,y) 获取像素
  2. isMatch = pixelMatchesColor(x,y,(r,g,b)) 像素匹配

3.图像定位

  1. locateOnScreen("goal_path") 可以获取希望单击的图像首次发现时左边的x坐标和顶点的y坐标，宽度w以及高度h（匹配算法一般，不支持背景变换等）
  2. locateCenterOnScreen("goal_path",grayscale=True,region=(x,y,w,h)，confidence=0.int) 定位并求中间点位置。grayscale可加速匹配，但也会增加错误概率。也可通过region指定查找范围。confidence可允许图像有轻微不同，可实现一定的模糊匹配。
  3. locateAllOnScreen("goal_path")能够在屏幕上找到多处匹配项。该函数会返回一个Generator对象，可以将他传递给list。

四、对话框

  1. ret = alert(text='text',title='title',button='alert') 创建对话框，单击按钮后返回button的值。
  2. confirm(text='text',title='title',button=['key_1','key_2','key_3']) ，创建带有多个按钮的对话框，单击按钮后返回该按钮的文本。
  3. prompt(text='text', title='title', default='请输入文本信息') 显示一个包含确认，取消的文本输入栏的信息框，用户可以指定输入内容。点击确认后会返回框中值，点取消会返回None。
  4. password(text='text', title='title', default='密码', mask='*') 与prompt类似，但输入的字符会显示为mask指定的符号。

五、窗口对象

属性：

  • left、 right、 top、 bottom 一个整数， 表示窗口边的x或y坐标。
  • topleft、 topright、 bottomleft、 bottomright 两个整数的命名元组， 表示窗口角的(x, y)坐标。
  • midleft、 midright、 midleft、 midright 两个整数的命名元组， 表示窗口边中间的(x, y)坐标。
  • width， height 一个整数， 表示窗口的一个维度， 以像素为单位。
  • size 两个整数的命名元组， 表示窗口的（宽度， 高度） 。
  • area 一个整数， 表示窗口的面积， 以像素为单位。
  • center 两个整数的命名元组， 表示窗口的中心（x, y） 坐标。
  • centerx、 centery 一个整数， 表示窗口中心的x或y坐标。
  • box 4个整数的命名元组， 表示窗口（左侧、 顶部、 宽度、 高度） 。
  • title 窗口顶部标题栏中的文本字符串。

获取窗口

  • getActiveWindow() 获取活动窗口
  • getAllWindows() 获取所有可见窗口
  • getWindowsAt(x,y) 获取所有包含点(x，y)的可见窗口的Windows对象列表
  • getWindowsWithTitle(title) 获取所有在标题栏中包含字符串title的可见窗口的Windows对象的列表。
  • getAllTitles() 获取所有可见窗口的字符串列表

操作窗口

  • win.move(xOffset, yOffset)
  • win.moveTo(newLeft, newTop)
  • win.resize(widthOffset, heightOffset)
  • win.resizeTo(width, height)
  • win.maximize()
  • win.minimize()
  • win.restore() # 如果是最大/小化，则恢复成正常状态
  • win.close()
  • win.position() # 获取窗口左上角坐标
  • win.moveRel(x=0, y=0) # 相对窗口左上角移动
  • win.clickRel(x=0, y=0, clicks=1, interval=0.0, button=’left’) # 相对窗口左上角点击

简单定位技术（mouseInfo）

概述： pyautogui.mouseInfo()函数可以单独使用，而不是作为程序的一部分。 它启动了一个名为MouseInfo的小应用程序， 该应用程序是PyAutoGUI的一部分。打开的窗口提供了关于鼠标指针当前位置的信息， 以及鼠标指针处的像素的颜色， 以3个整数的RGB元组和十六进制值的形式显示。 颜色本身会出现在窗口中的颜色框中。

  • Copy All、 Copy XY、 Copy RGB和Copy RGB Hex按钮将对应的信息复制到剪贴板上。
  • Log All、 Log XY、 Log RGB和Log RGB Hex按钮将对应的信息写入窗口中的大文本字段。
  • 可以通过单击Save Log按钮， 保存这个文本字段中的文本。
  • 可以通过使用 F1, F2 … F8 功能键类实现。
