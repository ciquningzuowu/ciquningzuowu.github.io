HTML速记表

发表时间：2026-03-11 阅读： 评论：

web 计算机基础

1. 基础结构

标签 描述 状态
<!DOCTYPE> 定义文档类型（HTML5 为 <!DOCTYPE html>） ✅ 现行
<html> 定义 HTML 文档的根元素 ✅ 现行
<head> 包含文档的元数据（标题、样式、脚本等） ✅ 现行
<body> 定义文档的主体内容（可见部分） ✅ 现行
<base> 指定页面上所有相对 URL 的基础 URL ✅ 现行
<link> 定义文档与外部资源（如 CSS 文件）的关系 ✅ 现行
<meta> 定义关于 HTML 文档的元数据（字符集、作者、描述等） ✅ 现行
<style> 定义文档的内部样式表 (CSS) ✅ 现行
<title> 定义文档的标题（显示在浏览器标签页上） ✅ 现行

2. 文本格式化

标签 描述 状态
<b> 粗体文本（无语义强调） ✅ 现行
<i> 斜体文本（通常表示术语、思想等） ✅ 现行
<u> 下划线文本（通常表示拼写错误或专有名词） ✅ 现行
<s> / <strike> 删除线文本 (<strike> 已废弃，用 <s>) ⚠️ <strike> 废弃
<strong> 重要文本（通常渲染为粗体，有语义强调） ✅ 现行
<em> 强调文本（通常渲染为斜体，有语义强调） ✅ 现行
<mark> 标记/高亮文本 ✅ 现行
<small> 小号文本（通常用于版权说明、侧注） ✅ 现行
<del> 被删除的文本 ✅ 现行
<ins> 被插入的文本 ✅ 现行
<sub> 下标文本 ✅ 现行
<sup> 上标文本 ❌ 现行
<code> 计算机代码文本 ✅ 现行
<kbd> 键盘输入文本 ✅ 现行
<samp> 计算机程序输出的样本 ✅ 现行
<var> 变量文本 ✅ 现行
<cite> 引用作品标题 ✅ 现行
<q> 短引用（浏览器通常会自动添加引号） ✅ 现行
<blockquote> 长引用（块级引用） ✅ 现行
<abbr> 缩写词 ✅ 现行
<address> 定义文档作者/拥有者的联系信息 ✅ 现行
<bdi> 文本双向隔离（防止周围文本影响方向） ✅ 现行
<bdo> 文本双向覆盖（强制文本方向） ✅ 现行
<ruby> 定义 ruby 注释（常用于东亚字符注音） ✅ 现行
<rt> 定义 ruby 注释的解释部分 ✅ 现行
<rp> 定义在不支持 ruby 时显示的括号 ✅ 现行
<time> 定义日期/时间 ✅ 现行
<wbr> 定义可能的换行点 ✅ 现行
<br> 单行换行 ✅ 现行
<hr> 水平分割线（主题分隔） ✅ 现行
<pre> 预格式化文本（保留空格和换行） ✅ 现行
<font> 定义字体、颜色和大小 ❌ 已废弃
<center> 居中对齐 ❌ 已废弃

3. 表单和输入 (Forms & Input)

用于收集用户输入。

表格

标签 描述 状态
<form> 定义用户输入的 HTML 表单 ✅ 现行
<input> 输入控件（类型由 type 属性决定） ✅ 现行
<textarea> 多行文本输入控件 ✅ 现行
<button> 可点击按钮 ✅ 现行
<select> 下拉列表 ✅ 现行
<option> 下拉列表中的选项 ✅ 现行
<optgroup> 下拉列表中的选项组 ✅ 现行
<label> 表单控件的标签 ✅ 现行
<fieldset> 围绕表单相关元素的边框 ✅ 现行
<legend> 定义 <fieldset> 的标题 ✅ 现行
<datalist> 定义输入控件的预定义选项列表 ✅ 现行
<output> 定义计算结果的输出 ✅ 现行
<progress> 进度条 ✅ 现行
<meter> 标量测量（如磁盘使用量） ✅ 现行
<keygen> 生成密钥对（已废弃，不再推荐使用） ❌ 已废弃
<isindex> 定义单行搜索框（极旧，已废弃） ❌ 已废弃

4. 框架 (Frames)

用于创建框架集（现代网页开发中已基本不再使用，被 iframe 或 CSS 布局取代）。

表格

标签 描述 状态
<frame> 定义框架集中的窗口 ❌ 已废弃
<frameset> 定义框架集 ❌ 已废弃
<noframes> 定义不支持框架时的替代内容 ❌ 已废弃
<iframe> 内联框架（在当前页面嵌入另一个页面） ✅ 现行

5. 图像 (Images)

用于嵌入图像和图形。

表格

标签 描述 状态
<img> 嵌入图像 ✅ 现行
<map> 定义图像映射（可点击区域） ✅ 现行
<area> 定义图像映射中的区域 ✅ 现行
<canvas> 绘制图形的画布（通常通过 JS 操作） ✅ 现行
<svg> 定义可缩放矢量图形 ✅ 现行
<figure> 规定自包含内容（如图片、图表、代码块） ✅ 现行
<figcaption> 定义 <figure> 的标题 ✅ 现行
<picture> 定义多个图像资源（用于响应式图片） ✅ 现行
<source> 定义媒体资源（用于 <video>, <audio>, <picture>） ✅ 现行

6. 音频/视频 (Audio & Video)

多媒体标签。

表格

标签 描述 状态
<audio> 定义声音内容 ✅ 现行
<video> 定义视频内容 ✅ 现行
<track> 定义视频/音频的文本轨道（如字幕） ✅ 现行
<embed> 嵌入外部应用程序或交互式内容（插件） ✅ 现行
<object> 定义嵌入对象（如 PDF, Flash, 插件） ✅ 现行
<param> 定义 <object> 的参数 ✅ 现行

7. 链接 (Links)

用于导航和超链接。

表格

标签 描述 状态
<a> 定义超链接 ✅ 现行
<nav> 定义导航链接的部分 ✅ 现行

8. 列表 (Lists)

用于组织列表数据。

表格

标签 描述 状态
<ul> 无序列表 ✅ 现行
<ol> 有序列表 ✅ 现行
<li> 列表项 ✅ 现行
<dl> 描述列表 ✅ 现行
<dt> 描述列表中的术语 ✅ 现行
<dd> 描述列表中术语的描述 ✅ 现行
<menu> 定义命令/菜单列表（现主要用于上下文菜单） ✅ 现行
<command> 定义用户可调用的命令（已废弃，功能并入 <menu>） ❌ 已废弃
<dir> 目录列表（已废弃，用 <ul> 代替） ❌ 已废弃

9. 表格 (Tables)

用于展示表格数据。

表格

标签 描述 状态
<table> 定义表格 ✅ 现行
<caption> 定义表格标题 ✅ 现行
<thead> 定义表格的表头部分 ✅ 现行
<tbody> 定义表格的主体部分 ✅ 现行
<tfoot> 定义表格的页脚部分 ✅ 现行
<tr> 定义表格行 ✅ 现行
<th> 定义表格标题单元格 ✅ 现行
<td> 定义表格标准单元格 ✅ 现行
<col> 定义表格列的属性 ✅ 现行
<colgroup> 定义表格中列的组 ✅ 现行

10. 样式、语义和布局 (Style, Semantic & Layout)

HTML5 引入的语义化标签，用于构建页面结构和样式控制。

表格

标签 描述 状态
<div> 块级容器（通用分组） ✅ 现行
<span> 行内容器（通用分组） ✅ 现行
<header> 定义文档或节的页眉 ✅ 现行
<footer> 定义文档或节的页脚 ✅ 现行
<main> 定义文档的主要内容 ✅ 现行
<section> 定义文档中的一个节（章节） ✅ 现行
<article> 定义独立的文章内容 ✅ 现行
<aside> 定义侧边栏或与主内容间接相关的内容 ✅ 现行
<details> 定义用户可展开/折叠的详情 ✅ 现行
<summary> 定义 <details> 的可见标题 ✅ 现行
<dialog> 定义对话框或窗口 ✅ 现行
<h1> - <h6> 定义标题（h1 最大，h6 最小） ✅ 现行
<p> 定义段落 ✅ 现行
<span> 定义行内片段 ✅ 现行
<class> (注意：这是属性不是标签，样式主要通过 class/id 配合 CSS) N/A
<acronym> 定义首字母缩写词（HTML5 已废弃，用 <abbr>） ❌ 已废弃
<applet> 定义 Java Applet（已废弃） ❌ 已废弃
<bgsound> 背景声音（IE 特有，已废弃） ❌ 已废弃
<blink> 闪烁文本（已废弃） ❌ 已废弃
<marquee> 滚动文本（已废弃） ❌ 已废弃
<spacer> 空白占位符（已废弃） ❌ 已废弃
<tt> 打字机文本（已废弃，用 <code> 或 CSS） ❌ 已废弃
<xmp> 预格式化文本（已废弃，用 <pre>） ❌ 已废弃

11. 元信息 (Metadata)

主要在 <head> 中使用，提供关于页面的信息。

表格

标签 描述 状态
<meta> 元数据（字符集、视口、SEO 描述等） ✅ 现行
<link> 外部资源链接（favicon, CSS） ✅ 现行
<title> 页面标题 ✅ 现行
<base> 基础 URL ✅ 现行
<noscript> 当浏览器不支持脚本或脚本被禁用时显示的内容 ✅ 现行
<script> 客户端 JavaScript 脚本 ✅ 现行
<template> 存储客户端模板内容（不立即渲染） ✅ 现行

12. 编程与脚本 (Programming & Scripting)

用于嵌入代码、插件或定义数据结构。

表格

标签 描述 状态
<script> 定义客户端脚本（JS） ✅ 现行
<noscript> 定义无脚本时的替代内容 ✅ 现行
<canvas> 用于通过脚本（通常是 JS）绘制图形 ✅ 现行
<embed> 嵌入外部交互内容（插件） ✅ 现行
<object> 嵌入外部对象（PDF, Flash 等） ✅ 现行
<param> 定义对象的参数 ✅ 现行
<applet> 嵌入 Java Applet（已废弃） ❌ 已废弃
<plaintext> 以纯文本形式显示剩余文档（极度危险，已废弃） ❌ 已废弃
