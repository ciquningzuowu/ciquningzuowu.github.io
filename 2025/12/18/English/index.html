英语基础补习笔记

基础句子结构(五大句型)

  • 主谓: 主语+不及物动词
  • 主谓宾: 主语+及物动词+宾语
  • 主谓双宾: 主语+及物动词+双宾语
  • 主谓宾补: 主语+及物动词+宾语+宾语补足语
  • 主系表: 主语+系动词+表语

修饰语

  • 冠词
  • 形容词
  • 副词
  • 介词短语
  • 非谓语动词
  • 定语从句
  • 状语从句
  • 等

谓语

1.动词 \begin{cases} 实义动词: \begin{cases} 不及物动词 \\及物动词 \end{cases} \\非实义动词:六大类(六大类都是起表示作用，是什么状态\approx变成什么状态\approx保持什么状态\approx似乎是什么状态\approx证实是什么状态) \begin{cases}be \\感官动词:feel、sound、taste、look、smell \\变化系动词:become、grow、tuen、get、go(均翻译为变得) \\表象系动词seem、appear \\持续系动词:keep、stay、remain、lie \\终止系动词:prove、turn out \end{cases} \end{cases}

2.谓语的五种状态:情态、时态、语态、否定、一致。句子中的这五个成分都属于谓语。

时态

时态为发生时间与运动状态的综合体， 发生时间分为现在、过去、将来、过去将来， 运动状态分为一般、进行、完成、完成进行。

一般现在时

句式转换:

  • 肯定句
  • 否定句:be动词后加not
  • 一般疑问句:be动词提前
  • 特殊疑问句:特殊疑问词+一般疑问句
  • 祈使句:be+其他

一般过去时

转换:be->was，were；v(s)->v(ed) 句式转换:

  • 否定句:主语+don'not+V/主语+did'not+其他
  • 一般疑问句Do/Does提前

现在/过去进行时

be/was/were+Ving

一般/过去将来时

will/shell+be/V would/should+be/V am/is/are going to+be/V

现在完成时

have/has+动词过去分词 {have/has+beanhave/has+Ved \begin{cases} have/has+bean\\ have/has+Ved \end{cases} {​have/has+bean​have/has+Ved​​ (两者为独立不共存的两种情况)

过去完成时

表示在过去某个动作或某个具体时间之前发生或完成的动作 had+动词过去分词 {had+beenhad+Ved \begin{cases} had+been\\ had+Ved \end{cases} {​had+been​had+Ved​​ (两者为独立不共存的两种情况)

将来进行时

表示将来某个时刻或某一段时间里正在发生的动作 shall/will+be+Ving

被动语态

一般现在时、一般现在时:

主语+（时态+语态）+其他(8BE+Ved，8BE指任意be动词:be、am、is、are、were、was、been、being)

一般将来时和过去将来时:

will/would be done

现在进行时和过去进行时:

am/is/are being done
was/were being done

现在/过去完成时:

have/has been done

情态

常见情态动词:can、may、must、shall、will 情态动词表示主语的情绪和态度，是助动词，必须帮助动词形成复合谓语。情态动词在时态和语态的前面。

词义角度: 情态动词有其自身的意义，来表示可能、建议、愿望、必要、允许、能力、怀疑等，以表示说话者对某种行为或状态的看法或意见

用法: 有三层含义:基本意思、推测、虚拟语气 情态动词后面+动词原形(时态+语态) 否定句not加在情态动词后面

形容词

形容词在名词短语中可以放在被修饰名词的前面或后面，于是构成了前置修饰和后置修饰，也叫做前置定语和后置定语(修饰不定代词是形容词要后置)

形容词的比较级

形容词有原级、比较级和最高级，一下是变换规则:(句子中的最高级部分前面要加the，本质原因是最高级的是可以确定的特定个体，用the特指) 1.单音节词:

  • 常规的单音节词直接加er和est变换
  • 以-e结尾的直接加r和st
  • 以-y结尾，y变i加er和est
  • 原因加辅音结尾，先双写词尾的辅音在加er和est 2.双音节词(两个元音)
  • 以y结尾，同上 3.特殊形式的比较级最高级 4.不具有等级的形容词 5.比较级的前面一般不加冠词the，但是当句子里有短语of the twe，比较级前面要加 6.用(more) and (more)表示越来越如何

形容词的最高级

最高级是把一个群体中的医院与整个群体进行比较 用法:

  • the+最高级+in+地方(范围、领域)
  • the+序数词+最高级+in+地方
  • the+形容词\副词最高级+of+所属范围

ing形容人的情况: 人的某种特质对外界(其他人)的影响。例:

  • the boring man 令人感到烦人的男人
  • the boing man 感到烦恼的男人

副词

可以修饰所有成分 按照意义分类: 按照功能分类:

副词的构成

1.一般情况下形容词加-ly 2.-y结尾将-y变为-ily

副词的位置

  • 程度副词一般放在被修饰的动词、形容词或副词前面。如果实义动词前面有情态动词或助动词要放在两者之间。如果是be动词，要放在其后。
  • 频率副词通常放在行为动词前，当句中有情态动词、助动词或be动词时，频率副词放在这类动词后面。

介词

只能接名词性成分(名词、代词、动名词、名词从句、名词短语)

介词短语的分类及用法功能

  • 做主语
  • 做宾补
  • 作表语
  • 做定语:介词短语对名词修饰要放在名词的后面，可以无限修饰

名词从句

作用上等效于名词，其构造为连接词+简单句 连接词: （1）that 不翻译 （2）whether/if 是否 （3）特殊疑问词 照常翻译

宾语从句

名词跟在动词或介词后做宾语从句

表语从句

名词从句充当表语对主语进行表示说明

主语从句

名词从句充当主语

同位语从句

名词解释名词（"我是奶龙"，奶龙解释我）

同位语从句=名词从句对抽象名词的解释说明。

同位语从句解释抽象名词的内容(定语从句修饰名词时作用为限定范围等，需区分) 常见抽象名词:

状语从句

  • 时间
  • 地点
  • 条件
  • 原因
  • 目的
  • 结果
  • 让步
  • 比较
  • 方式

本质是为主句服务的成分

时间状语从句

连接词：

  • when 一段时间，一点时间，一时性的动作，持续性的动作

  • as 指动作与主动作同时发生，具有延续的含义，强调一边。。。一边。。。

  • while 表示持续性动作或状态

  • before 表示引导的从句发生在主句动作之前。如果从句是过去式，主句一般要用过去完成时。

  • after 表示引导的从句发生在主句动作之后。如果主句是过去时，从句要用过去完成时

（若没有强调先后关系，则也可以用一般过去时）

  • until 直到。。。整体作用类似于yesterday或tomorrow，是一个服务于主句的时间修饰

地点状语从句

从句整体作副词性质，因为与地址连接词多为动词如停留（你停留并不完整，你停留在某处）

原因状语从句

连接词:

  • because（因为），
  • since（既然），
  • as（由于/既然接近since），
  • 连词for（表示推断的理由，
  • 对前面分局的内容加以解释说明），
  • in that（原因就在于）|引导的从句需要放在主句后面

目的状语从句

连接词：

  • so that
  • in order that
  • to
  • so as to +do

结果状语从句

  • such相当于so+形容词

条件状语从句

  • if如果
  • unless除非
  • as long as只要
  • so long as只要
  • providing假如
  • supposing假如
  • on condition that在。。。条件下
  • in case如果（以免）

让步状语从句

连接词：

  • though/although
  • even though
  • even if 虽然，尽管，即使等概念主句前不可用but，但可以用yet/still（这两个相当于副词而不是连词）

特殊引导词:

  • while尽管（位于句首时）

表示让步转折关系的介词和副词

  • despite即使
  • in spite of尽管
  • notwithstanding尽管 though等连接词后面只跟从句，而这里的介词只接名词（短语）

比较状语从句

as/so。。。as，前一个是副词，后一个才是比较状语从句的连词

方式状语从句

连接词：as和just as（好像/仿佛），引导的从句可以用陈述语气，表示所述情况是事实或实现的可能性，也可以用虚拟语气，表示不符合事实或与事实相反的情况。动词是be时，一般用were

定语从句

定义:属于对名词或代词的后置修饰，由关系代词或关系副词引导（通过从句的成分是否完整判断用代词还是副词） 定语:对名词或代词的修饰 这部分之前学的比较多不细作笔记

关系代词在从句作宾语时可以省略

定语从句分为限定性定语从句和非限定性定语从句（按照修饰信息是否为必须说明的信息，即如果没有从句部分句意无法理解，非限定性定语从句可以用逗号分出）

  • 特殊as引导非限定性定语从句
  • which引导的非限定性定语从句,可以指代和修饰主句的名词、代词、形容词、短语、其他从句、整句。在从句中作主语、动词宾语、介词宾语或表语
  • 只有as可用于句首。as更适合跟与主语一致或顺着主句的内容，而which则适合跟不顺应、反对、消极作用的内容

非谓语动词

重点:确定其逻辑主语是哪个，与逻辑主语的关系是什么 三种形式：

  • to do目的/要做|用于逻辑主语后，相当于形容词。也可作为句子主干，例如宾语、主语等
  • doing主动/进行|作主语、宾语、宾语补足语。作定语是相当于形容词，分词短语需要后置。做状语时需注意与逻辑主语的联系
  • done被动/完成|作定语时，分词短语后置，单独可以提前，也可后置起强调作用。作状语时表达逻辑主语的被动。

独立分词结构

结构：

  • 名词+现在分词做状语
  • with+名词+分词作状语

固定结构

  • 副词+speaking
  • 分词+介词
  • 分词+that

强调句

It is （强调部分） that （原句剩余部分）

  • 强调原因状语从句时只能用because引导
  • 被强调部分无论是单数还是复数，It后的动词均用单数形式，即is或was
  • 强调句的it is与that拆除后仍能组成正常句子

倒装句

出于语法或修饰效果，需要将主干顺序颠倒。

只是助动词、情态动词、系动词be提前称为部分倒装，整个谓语部分提前成为完全倒装

  • 否定副词或词组位于句首，剩余部分改为一般疑问句即为部分倒装
  • 提前only时需额外提前only后的条件

虚拟语气

虚拟语气是一种动词形式，用于表示说话人认为所说的内容并非客观事实，而是假设、愿望、建议、命令、怀疑或可能性极小的情况。 我们要虚构一个情景，在这个虚构的框架里讨论事情。既然是虚构的，就不能用描述现实的“陈述语气”，必须用一套专门标记“这不是真的”的动词形式，这就是虚拟语气。****

假设的时间点 If条件从句的谓语形式 主句的谓语形式 例句
与现在事实相反 动词的过去式 (be动词一律用were) would/should/could/might + 动词原形 If I were you, I would accept the offer.
(如果我是你，我会接受这个报价。)
与过去事实相反 had + 过去分词 (即过去完成时) would/should/could/might + have + 过去分词 If I had known the truth, I would have told you.
(如果我当初知道真相，我就会告诉你了。)
与将来事实可能相反 1. 过去式 (be用were)
2. should + 动词原形 (可能性极小)
3. were to + 动词原形 (纯粹的假设)		 would/should/could/might + 动词原形 1. If it rained tomorrow, we would cancel the trip.
2. If it should rain tomorrow, we would cancel the trip.
3. If it were to rain tomorrow, we would cancel the trip.
(如果明天下雨，我们就取消旅行。)
几个主要句型：
  • I wish。。。|我希望
  • How I wish。。。|我多么希望
  • If only。。。|要是。。。就好了
  • I would rather。。。|我宁愿。。。
  • It is high time that。。。|真的是到了某人该做某事的时候了

含蓄条件句

“含蓄条件句”是虚拟语气中非常高级且实用的用法。它不直接使用 if 引导的从句，而是通过介词短语、副词、上下文或其他方式来暗示一个“假设的条件”。 ![[{489B723D-75B3-4BAE-A943-0617E5D0B82B}.png]]
